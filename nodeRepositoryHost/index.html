This repository is a repository of programs which can be used on RandOS. RandOS is a python-based simulated operating system created by Luna because it got bored one time. If you would like to download RandOS, you first need to download the randos.py file from GitHub. You must then run the python file somwhere in your computer. An internet connection is required for the initial setup. After that, run the following command in the randos.py program:

sudo installer update

This command will set up the proper folder for downloading a repository. Next, download the latest version of this repository into ./cfg/proginstaller/repositories/ in the files folder. Now you can use our repository to download new programs.
